# Project Showcase App

A simple React app to showcase my projects. You can add, search, and delete projects.

---

## Features

- View a list of projects on the landing page
- Add new projects using the form
- Search projects by title
- Delete projects you no longer want

---

## How to run

1. Clone the repo
2. Run `npm install`
3. Run `npm start`
4. Open your browser at `http://localhost:3000`

---

## How to run tests

Run `npm test` in your terminal

---

## Folder structure

```
src/
  App.js
  App.test.js
  components/
    ProjectForm.js
    ProjectList.js
    SearchBar.js
```

---

## Known limitations

- Projects are not saved when you refresh the page
- No edit feature yet
